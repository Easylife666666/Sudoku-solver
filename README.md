Sudoku-solver

You need to prepare sudoku in the source code.
Then `gcc SudokuSolver.c` and `./a.out` is ok.

It’s a bit of a brute-force approach, but it gets the job done.
The current logic is suboptimal and leaves room for optimization.
Honestly, I have no plans to change it anytime soon!

100% Human-Coded (No LLM used in this project!)

Tested and fully working on Debian 13 (GCC 15), Ubuntu 4 (GCC 3 / 2.95), and Linux 0.11 (GCC 1.4).
